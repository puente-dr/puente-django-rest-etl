# puente-django-rest-etl
 
Create virtualenv
```bash
 python3 -m venv venv   
```

or
```bash
virtualenv venv --python=python3.8
```

Install requirements
```bash
source venv/bin/activate
pip3 install -r requirements.txt 
```


Run
```bash
python3 manage.py runserver 
```
